Google friendly FREE informative Sidebar Widgets for your WordPress Blogs. Helping Bloggers to make their Blogs or WebSites information rich. No registration required.

== Description ==

Free informative Sidebar Widgets to boost user activities. In these widgets links pointing to the Articles are Google Adsense friendly. User can read these Articles using 200+ Global Languages. All Articles showing under these Widgets are Family Safe.

After a successful installation you don't need to do anything about the look and feel, the plugin will automatically fit with your Site theme. Using Settings, you can Set the number of records, border and height of these Sidebar Widgets. Currently we are supporting 28 popular Categories. Depending upon your niche you can check all those Categories you need to display in the Sidebar.

You can display these Sidebar Widgets using Shortcode [i20_Sidebar_Widgets].

== Frequently Asked Questions ==

= How to Disable this Plugin? =

Just use standard Plugin overview page in WordPress admin section and deactivate it or rename plugin folder /wp-content/plugins/i20-sidebar-widgets over FTP access.

= Will it Slow my Site down? =

No, it won't. It's only loaded on the pages where Shortcode added.

== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `i20 Sidebar Widgets` and click 'Install Now' or click on the `upload` link to upload `i20-sidebar-widgets.zip`
4. Click on `Activate plugin`
